# 🔐 Modern Password Generator

A modern password generator built using **Python** and **Tkinter**. This application allows users to generate secure, customizable passwords through a clean and user-friendly graphical interface.

---

## ✨ Features

- 🔢 Custom password length (4–64 characters)
- 🔠 Uppercase letters
- 🔡 Lowercase letters
- 🔢 Numbers
- 🔣 Symbols
- 📋 One-click copy to clipboard
- 🖥️ Modern dark-themed graphical user interface
- 🚫 Error handling for invalid selections
  
---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- Random
- String

---

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/pranit-noaim/modern-password-generator.git
```

2. Open the project folder

```bash
cd modern-password-generator
```

3. Run the application

```bash
python password_generator.py
```
